# Canonical Identifiability of Hidden Fuzzy Relation Matrices from External Measurements

This repository provides reproduction code for the paper:

**Canonical Identifiability of Hidden Fuzzy Relation Matrices from External Measurements**

## Contents

- `gas_furnace_control_case_explicit_encdec.py`  
  Reproduction script for the explicit gas-furnace case study.

- `recovery_maxmin_fuzzy_relation_experiments.py`  
  Reproduction script for the synthetic experiments and learning-performance analysis.

- `gas_furnace_series.csv`  
  Raw Box–Jenkins gas-furnace time-series data used in the real-data case study.

## Description

The repository contains the code used to reproduce the main numerical experiments in the paper, including:

- recovery of hidden max–min fuzzy relation matrices from paired external measurements,
- synthetic experiments for identifiability and recovery performance,
- the gas-furnace real-data case study.

## Usage

Run the corresponding Python scripts directly after preparing the required Python environment.
